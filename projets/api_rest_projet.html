API - Gestion des Données

Ce projet consiste à développer une API RESTful permettant de gérer des ressources telles que des utilisateurs, des produits ou des commandes. L'objectif était de créer une interface accessible via HTTP pour manipuler ces ressources (CRUD).

Le projet utilise le framework Symfony et expose les données via des points d'accès sécurisés en utilisant des jetons d'authentification (JWT). Chaque requête peut être filtrée, triée et paginée pour une utilisation optimale côté client.

Les principales difficultés rencontrées ont été la gestion des autorisations et la mise en place des tests pour garantir la fiabilité de l'API.

Bilan des apprentissages

Ce projet m'a permis de renforcer mes compétences en développement d'API RESTful. J'ai appris à sécuriser une API avec des tokens JWT et à gérer les requêtes HTTP efficacement. L'intégration de tests avec PHPUnit a également été un point clé pour garantir le bon fonctionnement de l'API.
